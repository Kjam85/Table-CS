HTML Table Cheat Sheet

Tag Elements

Tags

Name

Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual data.
<tr> Table Row Table row container.
<td> Table Data Table row container.
<tfoot> Table Footer The set of rows defining the footer in a table.

Attribute Elements

Attribute

Name

Description
colspan Column Span The number of columns a <td> should span.
rowspan Row Span The number of rows a <td> should span.
